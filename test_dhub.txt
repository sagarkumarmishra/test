# 🎯 GOAL

Send data to DataHub using:

```text
Pipeline → Get Token → Pass Token → Notebook → Upload Data
```

---

# ✅ STEP 1 — Fix Web Activity (Token)

Go to:

👉 **Pipeline → GetApigeeToken (Web Activity)**

### Check these:

* Method = **POST**
* Headers:

```text
Content-Type : application/x-www-form-urlencoded
```

* Body:

```text
grant_type=client_credentials&client_id=YOUR_CLIENT_ID&client_secret=YOUR_CLIENT_SECRET
```

---

## ▶️ Run Debug once

👉 Go to **Output tab**

You should see:

```json
{
  "body": {
    "access_token": "eyJhbGciOi..."
  }
}
```

---

# ✅ STEP 2 — Pass token to Notebook

Go to:

👉 **Notebook activity → Settings → Base parameters**

### ❗ Delete old one (`bearer_token`)

### ➕ Add NEW:

| Name         | Value                                                |
| ------------ | ---------------------------------------------------- |
| access_token | @activity('GetApigeeToken').output.body.access_token |

---

# ✅ STEP 3 — Read token in Notebook

Open your notebook → add this **FIRST CELL**

```python
from notebookutils import mssparkutils

print("Reading token...")

token = mssparkutils.notebook.getParameter("access_token")

print("Token received:", token[:20])
```

# ✅ STEP 4 — Prepare data

Add next cell:

```python
import json
from datetime import datetime, timezone

data = {
    "eventTime": datetime.now(timezone.utc).isoformat(),
    "policyId": "POL123",
    "status": "ACTIVE",
    "amount": 1000
}

print("Data ready")
```

---

# ✅ STEP 5 — Send data to DataHub

Add next cell:

```python
import requests
import urllib3

urllib3.disable_warnings(urllib3.exceptions.InsecureRequestWarning)

url = "https://dhub-int.cta.allianz/api/v1/file/upload"

headers = {
    "Authorization": f"Bearer {token}",
    "Accept": "application/json",
    "tenant": "AZRE"
}

files = {
    "file": ("test.json", json.dumps(data), "application/json")
}

print("Sending data...")

response = requests.post(
    url,
    headers=headers,
    files=files,
    verify=False
)

print("Status:", response.status_code)
print("Response:", response.text)
```
