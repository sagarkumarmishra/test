### ✅ Correct Way to Read Parameters in Synapse Notebook (2026)

When you pass parameters from **Pipeline → Notebook Activity** (Base Parameters), use this:

```python
from notebookutils import mssparkutils

# Correct syntax
token = mssparkutils.notebook.params.get("bearer_token")
```

**Note:**  
- Use `.params.get("name")` → not `getParameter`  
- The `mssparkutils` is already available by default in Synapse Spark notebooks (you don't always need the import, but it's safe to keep it).
