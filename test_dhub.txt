# 🔍 Most likely root causes

## 1️⃣ Linked Service name is wrong

You used:

```python
"KeyVault001"
```

👉 But in Synapse, name must match **exactly**

Go check:

👉 **Manage → Linked services → Key Vault**

Maybe actual name is like:

* `LS_KeyVault`
* `KeyVault_LS`
* `KV_Allianz`

---

## 2️⃣ No permission to Key Vault

Spark needs access via:

👉 Managed Identity OR Service Principal

If missing → request hangs

---

## 3️⃣ Network issue

If Key Vault is private:

👉 Spark must be inside VNet
👉 Or private endpoint must exist
