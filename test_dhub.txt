### Why this keeps happening
In **Azure Synapse Spark Notebooks**, when you run a cell **directly** (by clicking the play button or "Run all"), the pipeline parameters are **not injected**.  
`mssparkutils.notebook.params.get()` only works when the notebook runs **through the Pipeline Notebook Activity** (via Debug or Trigger).

Your previous attempts failed for the same reason — the notebook was executed interactively.

### Correct & Simple Way (Recommended)

**Step 1: Create a "Parameters Cell"** (This is the official Microsoft way)

1. In your notebook, create a **new cell at the very top**.
2. Paste this exact code in that cell:

```python
# Parameters cell - DO NOT delete or move this cell
bearer_token = ""   # This will be overwritten by the Pipeline
```

3. Click on the cell → look for the **ellipses (...)** on the right → select **"Toggle parameter cell"** (or "Make parameters cell").

   - The cell background will change (usually light blue or marked as Parameters).

**Step 2: Use the parameter in your main code cell** (below the parameters cell)

Replace your current code with this full working version:

```python
import json
from datetime import datetime
import requests

print("=== Starting DataHub Upload ===")

# Token comes from the Parameters cell (injected by Pipeline)
token = bearer_token   # ← This variable is automatically available

print("Token received (first 40 chars):", 
      token[:40] + "..." if token and len(token) > 40 else "EMPTY or MISSING!")

if not token or len(token) < 50:
    raise Exception("❌ Token is missing! Run this notebook from the Pipeline (Debug/Trigger), not directly.")

# Prepare data
data = {
    "eventTime": datetime.utcnow().strftime("%Y-%m-%dT%H:%M:%SZ"),
    "policyId": "POL123",
    "status": "ACTIVE",
    "amount": 1000
}

json_content = json.dumps(data)

print("Data prepared:", json_content)

# Upload
url = "https://dhub-int.cta.allianz/api/v1/file/upload"

headers = {
    "Authorization": f"Bearer {token}",
    "Accept": "application/json",
    "tenant": "AZRE"
}

files = {
    "file": ("test.json", json_content, "application/json")
}

print("Sending upload request...")

response = requests.post(url, headers=headers, files=files, timeout=60)

print("Status Code:", response.status_code)
print("Response:", response.text[:800])   # show first 800 chars

if response.status_code == 200:
    print("✅ SUCCESS: Uploaded to DataHub!")
else:
    raise Exception(f"Failed with {response.status_code}: {response.text}")
```
