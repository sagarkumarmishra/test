# 🎯 GOAL

Send JSON data to DataHub using:

```text
Pipeline → Web Activity (Token) → Notebook → DataHub API
```

---

# ✅ STEP 1 — Configure Web Activity (Token)

Go to **Pipeline → GetApigeeToken (Web Activity)**

### Settings:

* **URL** → (already configured by DevOps)
* **Method** → `POST`
* **Body**:

```json
grant_type=client_credentials&client_id=YOUR_CLIENT_ID&client_secret=YOUR_CLIENT_SECRET
```

* **Headers**:

```json
Content-Type: application/x-www-form-urlencoded
```

---

## ✅ Test it

Click **Debug**

👉 Output should contain:

```json
{
  "access_token": "eyJhbGciOi..."
}
```

---

# ✅ STEP 2 — Pass token to Notebook

Click **Notebook activity (Notebook1)**

Go to **Settings → Base parameters**

Add:

| Name         | Value                                           |
| ------------ | ----------------------------------------------- |
| access_token | @activity('GetApigeeToken').output.access_token |

---

# ✅ STEP 3 — Read token in Notebook

Open your notebook → add this **first cell**

```python
from notebookutils import mssparkutils

print("Reading token...")

token = mssparkutils.env.getJobParameter("access_token")

print("Token received")
```

---

## ⚠️ Quick check

If token prints `None` → parameter mapping issue

---

# ✅ STEP 4 — Prepare data

Add next cell:

```python
import json
from datetime import datetime, timezone

data = {
    "eventTime": datetime.now(timezone.utc).isoformat(),
    "policyId": "POL123",
    "status": "ACTIVE",
    "amount": 1000
}

print("Data ready")
```

---

# ✅ STEP 5 — Send data to DataHub

Add next cell:

```python
import requests
import urllib3

urllib3.disable_warnings(urllib3.exceptions.InsecureRequestWarning)

url = "https://dhub-int.cta.allianz/api/v1/file/upload"

headers = {
    "Authorization": f"Bearer {token}",
    "Accept": "application/json",
    "tenant": "AZRE"
}

files = {
    "file": ("test.json", json.dumps(data), "application/json")
}

print("Sending data to DataHub...")

response = requests.post(
    url,
    headers=headers,
    files=files,
    verify=False
)

print("Status:", response.status_code)
print("Response:", response.text)
```

# 🚨 Troubleshooting (quick)

### ❌ 401

→ Token not passed correctly
→ Check Step 2

---

### ❌ None token

→ Wrong parameter name
→ Must be:

```text
access_token
```

---

### ❌ 403

→ Permission issue (team side)
