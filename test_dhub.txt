This is a **very common frustration** in Synapse. The reason it didn't work earlier is usually one (or more) of these small things:

### Why the token is not injected automatically

The pipeline **only** injects the token when:
- There is a **Parameters cell** at the **top** of the notebook.
- That cell is **correctly toggled** as "Parameters cell" (you see the "Parameters" label).
- The variable name in the cell (`bearer_token = ""`) **exactly** matches the **Name** you set in the Pipeline's Notebook Activity (case-sensitive: `bearer_token`).
- You run the notebook **only from the Pipeline** (Debug or Trigger), **not** by clicking Run all in the notebook itself.

If any of these is slightly off, the token remains empty → your code raises an error → Spark session gets **killed** (that's why your pipeline showed "Succeeded" but the notebook state was "killed/cancelled").

### Let's fix it cleanly right now (Most Reliable Method)

**Step 1: Parameters Cell (Top-most cell)**

Create/replace the **very first cell** with exactly this:

```python
# === PARAMETERS CELL - DO NOT DELETE OR MOVE ===
bearer_token = ""   # Pipeline will overwrite this with real Apigee token
```

- Click the **...** on the right of this cell.
- Choose **"Toggle parameter cell"**.
- Confirm you see **"Parameters"** label on the cell.

**Step 2: Main Code Cell (Put this in the second cell)**

```python
import json
from datetime import datetime
import requests

print("=== DataHub Upload Started ===")

# Get token injected by Pipeline
token = bearer_token.strip() if bearer_token else ""

print("Token length:", len(token))
print("Token preview:", token[:60] + "..." if len(token) > 60 else token)

if len(token) < 100:
    raise Exception("""❌ Token is empty!
1. Confirm Pipeline Notebook Activity has Base Parameter:
   Name = bearer_token   (exactly this spelling)
   Value = @activity('GetApigeeToken').output.token
2. You toggled the first cell as 'Parameters cell'
3. You are running via Pipeline Debug/Trigger (not direct Run all)
""")

# === Prepare data ===
data = {
    "eventTime": datetime.utcnow().strftime("%Y-%m-%dT%H:%M:%SZ"),
    "policyId": "POL123",
    "status": "ACTIVE",
    "amount": 1000
}

json_content = json.dumps(data)

print("Data being uploaded:", json_content)

# === Upload ===
url = "https://dhub-int.cta.allianz/api/v1/file/upload"

headers = {
    "Authorization": f"Bearer {token}",
    "Accept": "application/json",
    "tenant": "AZRE"
}

files = {
    "file": ("test.json", json_content, "application/json")
}

print("Sending multipart request...")

response = requests.post(url, headers=headers, files=files, timeout=120)

print("Status Code:", response.status_code)
print("Full Response:", response.text)

if response.status_code == 200:
    print("✅ SUCCESS: File uploaded to DataHub!")
else:
    raise Exception(f"Upload failed with {response.status_code}: {response.text}")
```
